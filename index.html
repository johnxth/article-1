Belajar GitHub Pages dengan Bootstrap

Cara mudah membuat website statis menggunakan GitHub Pages dan Bootstrap

Apa itu GitHub Pages?

GitHub Pages adalah layanan gratis dari GitHub yang memungkinkan Anda untuk menerbitkan situs web statis langsung dari repositori GitHub Anda.

Cara Menggunakan GitHub Pages

  1. Buat repositori baru di GitHub.
  2. Tambahkan file HTML, CSS, dan JavaScript ke repositori.
  3. Masuk ke pengaturan repositori dan aktifkan GitHub Pages.
  4. Pilih branch dan folder untuk dipublikasikan.
  5. Situs Anda akan tersedia di URL yang diberikan oleh GitHub.
